Instructions for sending files

  • Click the "SEND FILE" button to open the send page.
  • Choose a file using the file chooser or paste text into the input area.
  • Click the "Generate" / "Create" button to produce a QR code or a share link, then have the receiver scan it.

Instructions for receiving files

  • Click the "RECEIVE FILE" button to open the scanner page.
  • Use your device camera to scan the sender's QR code or open the shared link.
  • Follow the on-screen prompts to download and save the received file.
